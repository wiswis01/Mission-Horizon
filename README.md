# Mission-Horizon
A dashboard that helps overseas military members coordinate operations, ressources and people.

Mission-Horizon replaces fragmented SharePoint workflows and manual spreadsheet tracking with a unified, intelligent dashboard designed for the realities of CAF deployment screening and international posting management.

---

## The Problem

Today, CAF administrative officers manage deployment screening across dozens of members using disconnected tools — SharePoint lists, Excel trackers, and manual follow-ups. The result is missed compliance deadlines, limited visibility into member readiness, and no early warning system for benefit claim anomalies.

---

## What Mission-Horizon Does

- **Member Journey Timeline** — A real-time, visual timeline of each member's screening progress, with clear status indicators from initial screening through to deployment and relocation.
- **Kanban Screening Pipeline** — A drag-and-drop pipeline replacing SharePoint workflows, filterable by screening phase.
- **Global Readiness Map** — An interactive map showing posting locations worldwide, with readiness status at a glance.
- **Domain Compliance Dashboard** — Five-domain compliance tracking with progress indicators, overdue flags, and blocker alerts.
- **ML Anomaly Detection** — A statistical anomaly review queue that flags suspicious benefit claims before they become audit findings.
